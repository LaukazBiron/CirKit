# CirKit
Simulador de circuitos por diseño, por C09676

Una calculadora para análisis de circuitos eléctricos que permite calcular:
- Ley de Ohm
- Resistencias en serie y paralelo
- Potencia eléctrica

## Características
- Interfaz simple
- Cálculos precisos
- Soporte para múltiples componentes

## Uso
```python
python calculadora.py
